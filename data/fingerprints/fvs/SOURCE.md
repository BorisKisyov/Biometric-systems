# Sample fingerprint source

These sample fingerprint images were downloaded on 2026-04-14 from the public
`FVS dataset` described here:

- Curated dataset index:
  [robertvazan/fingerprint-datasets](https://github.com/robertvazan/fingerprint-datasets)
- Original project page:
  [Fingerprint Verification System](https://fvs.sourceforge.net/)
- Direct archive used:
  [fingerprint_bitmaps.zip](https://fvs.sourceforge.net/fingerprint_bitmaps.zip)

Dataset notes from the curated index:

- 21 fingers
- 8 impressions per finger
- BMP format
- 256x256 pixels
